Relay Station

Complete message relay workflow: cipher, encrypt, send, retrieve, decrypt, and decode

Step 1: Apply Cipher

Transform your plaintext message using classical cryptography methods.

Enter a number between 1-25 for Caesar cipher

Step 2: Modern Encryption

Secure your ciphered message using AES-256-GCM encryption with password-based key derivation.

This password will be needed to decrypt the message later

Step 3: Send to Relay Station

Upload your encrypted message to the relay station and receive a ticket number for retrieval.

Step 4: Retrieve from Station

Use your ticket number to retrieve the encrypted message from the relay station.

This should be automatically filled from Step 3

Step 5: Decrypt Message

Decrypt the retrieved message using the same password from Step 2.

Must match the encryption password exactly

Step 6: Decode Cipher

Reverse the cipher from Step 1 to reveal your original plaintext message.

Use the same key from Step 1
