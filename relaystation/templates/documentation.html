Project Overview

CipherStation is a comprehensive cryptography toolkit built as my final project for CS50x. It combines modern cryptographic algorithms with classical cipher analysis, advanced statistical pattern recognition, and a secure message relay system. The project demonstrates advanced concepts in cryptography, statistical analysis, web development, and secure software design.

Modern Cryptography

AES-256-GCM, ChaCha20-Poly1305, Ed25519 signatures, X25519 key exchange, Argon2id key derivation, and hybrid encryption.

Classical Cipher Analysis

Caesar, Vigenère, XOR, Atbash, and Substitution ciphers with advanced statistical analysis and 500+ word dictionary attacks.

Advanced Statistical Analysis

Multi-dimensional pattern recognition, frequency analysis, bigram/trigram analysis, and confidence scoring algorithms.

Secure Message Relay

Ticket-based dead drop system with automatic 24-hour cleanup for anonymous encrypted message exchange.

Comprehensive CLI

Full-featured command-line interface with interactive menu, bulk operations, and advanced cryptanalysis capabilities.

Security & Audit

Hash-chained audit logging, key registry management, format detection, and comprehensive self-tests.

System Architecture

Core Components

  • ciphercore.py - Core cryptographic functions and algorithms
  • app.py - Flask web application and API endpoints
  • templates/ - HTML templates with responsive design
  • static/ - CSS, JavaScript, and static assets
  • tests/ - Comprehensive test suites for all functionality
  • cli_cracker.py - Advanced CLI cipher cracking interface
  • advanced_cryptanalysis.py - Advanced statistical analysis and genetic algorithms
  • classical_ciphers.py - Classical cipher implementations

Data Flow

  1. User enters message and password
  2. Password is converted to appropriate key length using Argon2id
  3. Message is encrypted using selected algorithm
  4. Encrypted data is stored with salt for key reconstruction
  5. Message can be sent to relay station or copied directly
  6. Decryption reverses the process using the same password
  7. Classical cipher analysis uses statistical pattern recognition
  8. Advanced algorithms employ genetic algorithms and multi-dimensional analysis

Supported Cipher Types

  • Caesar Cipher: Shift-based substitution (key: 0-25)
  • Vigenère Cipher: Keyword-based polyalphabetic substitution
  • XOR Cipher: Bitwise XOR operation with key
  • Atbash Cipher: Simple letter reversal (no key needed)
  • Substitution Cipher: Custom 26-letter substitution key
