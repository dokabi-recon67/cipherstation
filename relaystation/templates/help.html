Usage Guide
Encrypting a File
  1. Click "Choose File" in the Encrypt section
  2. Select your file (max 20MB)
  3. Optionally upload a key file (or let us generate one)
  4. Choose your encryption algorithm
  5. Add an optional description
  6. Click "Encrypt File"
  7. Download your encrypted file
Decrypting a File
  1. Click "Choose File" in the Decrypt section
  2. Select your encrypted file (.enc.json)
  3. Upload the key file used for encryption
  4. Click "Decrypt File"
  5. Download your decrypted file
Message Encryption Usage
Quick Message Encryption
  1. Enter your message in the left text box
  2. Provide an encryption key
  3. Select your preferred algorithm
  4. Click "Encrypt" to encrypt locally
  5. Or click "Send to Station" to store encrypted message
  6. Use the copy button to copy encrypted text
Message Station
  1. Send encrypted messages to the station
  2. Receive a ticket number for each message
  3. Share the ticket number with recipients
  4. Recipients can search by ticket number
  5. Use the decryption form to decrypt messages

Security Information
Cryptographic Standards
  • AES-GCM: NIST-approved authenticated encryption
  • ChaCha20-Poly1305: RFC 8439 standard
  • Key Derivation: Argon2id for password-based keys
  • Random Generation: Cryptographically secure RNG
Privacy & Data Handling
  • Files are processed in temporary memory only
  • No files or keys are stored on servers
  • All processing happens in your browser
  • Message station stores only encrypted data
Command Line Interface

For advanced users, CipherStation provides a full-featured CLI with additional capabilities:

  • File Operations: Encrypt/decrypt files of any size
  • Digital Signing: Ed25519 signature generation and verification
  • Hybrid Encryption: X25519 key exchange with AES-GCM
  • Batch Processing: Encrypt multiple files at once
  • Key Management: Generate and manage key pairs
