Installation Instructions
From GitHub

Clone the repository and install dependencies:

git clone https://github.com/dokabi-recon67/cipherstation.git
cd cipherstation
pip install -r requirements.txt
Direct Download

Download the single-file version:

Download cipherstationv0.py Single file, no dependencies to install
Usage Examples
Generate Keys
# Generate AES key
python cipherstationv0.py keygen aes256 --out mykey.key

# Generate Ed25519 keypair
python cipherstationv0.py keygen ed25519 --priv private.key --pub public.key
Encrypt/Decrypt
# Encrypt file
python cipherstationv0.py encrypt --key mykey.key --infile secret.txt --out encrypted.json

# Decrypt file
python cipherstationv0.py decrypt --key mykey.key --infile encrypted.json --out decrypted.txt
Advanced Features
Password-Based
python cipherstationv0.py derive --out key.key
Sign Files
python cipherstationv0.py sign --priv private.key --infile doc.txt --sig signature.json
Batch Encrypt
python cipherstationv0.py encrypt-dir --key key.key --in-dir docs --out-dir encrypted
